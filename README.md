# Blender donut and coffee
First project, following Blender Guru's Beginner Blender Tutorial (2026) on YouTube. Contains Blender file, a PBR texture for the wooden table, Blender Guru's Add-On pack, and a reference image for the mug.

Includes: 
- Blender scene with a donut, plate, coffee mug, and table
- Reference photo for creating the mug
- Texture pack and add-ons for the table, donut, and plate textures
- STL of the Blender scene
